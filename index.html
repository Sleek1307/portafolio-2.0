Mis Tecnologías

  • HTML

    El lenguaje de marcado de hipertexto es aquella herramienta que se usa para estructurar una pagina web, la pagina fuera una casa esta serían sus columnas y cimientos

  • CSS

    Es el lenguaje que se usa para dar estilo a los elementos de la pagina, siguiendo con el ejemplo de la casa este seria la pintura y los acabados de la fachada

  • JavaScript

    Es el lenguaje que usan los navegadores para darle funcionalidad a las paginas, este permite interacuar con los diversos elementos de la pagina y darles funcionalidad, volviendo al ejemplo de la casa este seria quien se encarga de los servicios como electricidad, agua y gas.

  • Bootstrap

    Es una libreria de CSS que ya trae unos estilos pre definidos y que nos permite dar estilo a la pagina de una manera mas sencilla, mediante una serie de clases pre definidas

  • ReactJs

    Libreria de JavaScript que nos permite modificar la pagina de una manera mucho mas sencilla, combinando HTML con JavaScript, permitiendo asi que la pagina sea mucho mas dinamica

  • Express

    Libreria de JavaScript que corre en el servidor y que sirve para construir backends usando el mismo lenguaje que se usa en el navegador

  • Laravel

    Libreria de PHP construida con la misma intencion de Express
